# Smart Contract Documentation

## Contract Purpose

The smart contract manages the donation system on the Ethereum blockchain. It ensures that all donations are recorded in an immutable and transparent way.

## Key Functions

### donate()

This function allows users to send ETH to the donation contract.

Parameters:
- Donation amount

Functionality:
- Accepts ETH
- Records the donation
- Emits a transaction event

## Benefits of Smart Contracts

- Transparency
- Security
- Decentralization
- Trustless transactions
