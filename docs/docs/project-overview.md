# Transparent Donation Tracker

## Overview

The Transparent Donation Tracker is a Web3 decentralized application (dApp) designed to improve transparency and accountability in charitable donations. Traditional donation systems often lack transparency, making it difficult for donors to verify how funds are handled.

This platform uses blockchain technology to record donations permanently on the Ethereum blockchain. Each donation transaction is publicly verifiable using its transaction hash.

## Objectives

The objectives of this project include:

- Providing transparency in donation systems
- Recording donation transactions on blockchain
- Allowing public verification of donations
- Building a decentralized application using Web3 technologies

## Technologies Used

Frontend:
- React
- Vite
- Axios

Backend:
- Node.js
- Express.js
- Ethers.js

Blockchain:
- Solidity Smart Contract
- Ethereum Test Network

## Key Features

- Blockchain-based donation tracking
- MetaMask wallet connection
- Real-time donation display
- Transaction verification through Etherscan
- Donation goal progress tracking
