# Transparent Donation DApp

A blockchain-based donation tracking system designed to ensure transparency, accountability, and trust in charitable contributions.

---

## 📌 Project Overview

This project is a decentralized application (DApp) that allows users to make donations in ETH while ensuring that every transaction is recorded on the blockchain. The system eliminates ambiguity in fund management by providing a transparent and immutable ledger of all donations.

---

## 🎯 Objectives

- To promote transparency in donation systems  
- To leverage blockchain technology for accountability  
- To provide a secure and tamper-proof donation record  
- To build a simple and user-friendly donation interface  

---

## 🛠️ Technologies Used

- **Frontend:** React (Vite)  
- **Backend:** Node.js + Express  
- **Blockchain:** Ethereum (Hardhat)  
- **Smart Contracts:** Solidity  
- **Web3 Integration:** Ethers.js / Axios  

---

## ⚙️ System Architecture

The application is divided into three main components:

### 1. Frontend
- User interface for making donations  
- Sends donation requests to the backend  

### 2. Backend
- Handles API requests  
- Interacts with the blockchain  
- Sends transactions using a private key  

### 3. Smart Contract
- Deployed on a local Ethereum network (Hardhat)  
- Records all donation transactions permanently
